aws_ec2 Cookbook
================
AWS EC2 cookbook.

Attributes
----------
Nothing

Usage
-----
#### aws_ec2::default

e.g.
Just include `aws_ec2` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[aws_ec2]"
  ]
}
```
